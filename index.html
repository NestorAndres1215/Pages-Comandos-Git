Comandos Esenciales de Git

Guía completa para el control de versiones

git init Inicialización

Inicializa un nuevo repositorio Git en el directorio actual.

git init
git clone Repositorio

Clona un repositorio existente desde una URL remota.

git clone https://github.com/usuario/repo.git
git status Inspección

Muestra el estado del árbol de trabajo, incluyendo cambios rastreados y no rastreados.

git status -s
git add Staging

Añade cambios del directorio de trabajo al área de staging.

git add archivo.txt || git add .
git commit Guardado

Guarda los cambios del área de staging en el historial del repositorio. 

git commit -m "Mensaje descriptivo"
git push Sincronización

Envía los commits locales a un repositorio remoto.

git push origin main
git pull Sincronización

Obtiene e integra cambios de un repositorio remoto.

git pull origin main
git branch Ramas

Lista, crea o elimina ramas.

git branch nueva-rama
git checkout Ramas

Cambia entre ramas o restaura archivos del árbol de trabajo.

git checkout -b nueva-rama
git merge Ramas

Combina el historial de ramas.

git merge feature-branch
git stash Temporal

Guarda temporalmente cambios sin commit.

git stash pop
git log Inspección

Muestra el historial de commits con diversas opciones de formato.

git log --oneline --graph --all
git reflog Rescate

Muestra un registro de todos los movimientos de HEAD (útil para recuperar commits perdidos).

git reflog
git revert Deshacer

Crea un nuevo commit que deshace cambios de un commit anterior (más seguro que reset).

git revert abc123
git config alias Configuración

Crea atajos personalizados para comandos frecuentes.

git config --global alias.st "status -s"
git blame Inspección

Muestra quién modificó cada línea de un archivo y en qué commit.

git blame archivo.txt
git show Inspección

Muestra información detallada de un commit específico.

git show abc123
git fetch --prune Sincronización

Sincroniza ramas remotas y elimina las que ya no existen.

git fetch --prune
git shortlog Inspección

Muestra un resumen de commits por autor.

git shortlog -sn
git rebase Ramas

Reaplica commits encima de otra rama base.

git rebase main
git reset Deshacer

Deshace commits o cambios en el área de staging.

git reset --hard HEAD~1
git rebase -i Ramas

Reorganiza, edita o combina commits interactivamente (¡cuidado con esto en ramas públicas!).

git rebase -i HEAD~3
git cherry-pick Ramas

Aplica un commit específico de otra rama a tu rama actual.

git cherry-pick abc123
git bisect Depuración

Ayuda a encontrar el commit que introdujo un bug usando búsqueda binaria.

git bisect start git bisect bad git bisect good v1.0
git worktree Productividad

Permite trabajar con múltiples ramas en directorios separados simultáneamente.

git worktree add ../feature-ruta feature/ruta
git push --force Sincronización

Fuerza el push sobrescribiendo el remoto (¡puede borrar trabajo de otros!).

git push --force origin main
git gc Mantenimiento

Optimiza el repositorio (recolector de basura).

git gc --aggressive
git fsck Mantenimiento

Verifica la integridad del sistema de archivos Git.

git fsck --full
git submodule Avanzado

Gestiona repositorios anidados dentro de tu proyecto.

git submodule update --init --recursive
